代码题
1 代码思路是构建3个集合
集合1： 新字典有，而旧字典没有的key
集合2： 老字典有，而新字典没有的key
集合3： 新老字典都有的key

通过分别迭代这3个集合，将集合1的key添加入老字典，将集合2的key从老字典删除，将集合3的key跟新到老字典

2 构建一个保存结果的字典
result = {"num": [0, ""], "alpha": [0, ""], "blank": [0, ""], "other": [0, ""]}
通过迭代传入对象的每个元素，通过条件判断知道元素是数字，字符，空格，还是其他字符，并将数量变更到构建的result字典中，最后
打印出字典的结果

3~7 题的思路大致相同，代码分为自定义函数部分和main主体部分，函数部分用于完成题目的需求，而主体部分通过循环，与用户交互，
获取用户输入的内容，并求值得到python对象。将python对象传入到定义的需求函数中进行解析。
其中题6中使用到了列表生成式，能够用一行代码简单的实现原本需要多行函数定义才能实现的功能
ret = [obj[x] for x in range(len(obj)) if x % 2 == 1]

8 定义了斐波那契函数fib(n)，其中当n为0或者1时，函数直接可以返回定义好的基准值，而当n>2时，通过递归的方式逐层调用函数自身，得到最后的结果
